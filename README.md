# game10003-a1-interactive-terminal-project

This is the repository for assignment 1 of game10003, that being the Interactive Terminal Project.



Instructions:

In this program, you are introduced to the virtual grocery store Digimart. You will first go to the dairy section where you

will be given the option to buy either milk or cheese by typing in the appropriate item, then you will repeat the process with meat.

Once this is over, you will be given a final total for how much money you have left over.
